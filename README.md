# Customer Churn SQL Analysis


## Project Overview
Analyzed telecom customer churn data using SQL to identify 
which customer segments are most likely to cancel service.


## Key Question
Which factors — contract type, tenure, and monthly charges —
best predict customer churn?


## Tools Used
- SQL (SQLite via sqliteonline.com)
- Dataset: IBM Telco Customer Churn (Kaggle, 7,043 rows)


## Key Findings
1. Overall churn rate: 26.5%
2. Month-to-month contract customers churn at 42% vs 3% for
   two-year contracts
3. New customers (0-12 months) have the highest churn rate
4. Higher monthly charges correlate with higher churn
